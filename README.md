# Glimpse - Social Media Platform

A beautiful, modern social media app built with React + Vite (Frontend) and Node.js + Express + MongoDB (Backend).

## Features
- User authentication
- Posts, Moments, Chat
- Real-time likes & follows
- Beautiful UI with glassmorphism & gradients
